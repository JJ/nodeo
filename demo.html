NodEO evolutionary algorithm demo

This webpage includes the code of NodEO, a JavaScript library for evolutionary algorithms. It starts running it when it loads. You can reload to start it all over again.

The evolutionary algorithm tries to find the 1s and 0s string that maximizes a certain function. In this case, maximum would correspond to a string filled with ones, only the algorithm does not know that in advance so it must try, change and combine mimicking natural selection until it finds the solution.
